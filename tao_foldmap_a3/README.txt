Tao Folding Map
README
-------------------------------

Folding Map lets you open a tactical map while in game which is bigger and more useful than the GPS.

HOW TO
-------------------------------
Press Shift-M (or Shift-<your Open Map key>) to pop up the map.

While the map is open, press Shift-Left Arrow and Shift-Right Arrow to change its screen location.
Press Shift-Up Arrow to recenter the folding map.
